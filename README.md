## Deploy to GitHub Pages (FREE)

### Step 1 — Create GitHub account
Go to [github.com](https://github.com) and sign up for free.

### Step 2 — Create a new repository
- Click "New repository"  
- Name it: `rupeesmart` (or any name)  
- Set to **Public**  
- Click "Create repository"

### Step 3 — Upload files
- Click "uploading an existing file"  
- Drag and drop ALL files from this folder  
- Click "Commit changes"

### Step 4 — Enable GitHub Pages
- Go to repository → **Settings** → **Pages**
- Source: "Deploy from a branch"
- Branch: **main** → folder: **/ (root)**
- Click Save

### Step 5 — Your site is live!
After 1-2 minutes, your site will be at:  
`https://YOUR-USERNAME.github.io/rupeesmart/`

## Add Google AdSense (after approval)

1. Sign up at [adsense.google.com](https://adsense.google.com)
2. Add your GitHub Pages URL as your site
3. Wait for approval (usually 2-4 weeks for finance sites)
4. Replace all `[Google AdSense — 728×90 Leaderboard]` placeholders with your actual ad code

### Where to place ads for maximum revenue:
- **Top of page** (leaderboard 728×90): After the nav, before article content  
- **Mid-article** (rectangle 300×250): After 2nd or 3rd section of every article  
- **Sidebar** (rectangle 300×250): Sticky sidebar on article pages  
- **End of article** (large rectangle): Before "read next" section
